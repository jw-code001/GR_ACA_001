데이터베이스 포멧에 대한 이해가 있는 경우
   - 폼양식을 관리하고자한다
   - 폼관련 테이블을 만들어야지
   - 운영관리에 대한 결정을 내려야한다.

* 질문번호는 pk이다.
  이전 데이터를 위해 수정은 없고 삽입만있음
  데이터타입은 관리대상이므로 저장하고 쓰자
  질문을 출력여부를 선택할 수 있는 컬럼과 생성날짜

////////////////////////////
   1. 수파베이스테이블을 생성한다.
   2. 수파베이스 테이블 스키마를 캡쳐떠서 
      LLM 더미데이터 만들어달라고 한다.  체크하기
   3. 구글앱스크립트 만들어달라고 한다. ( 질문 )
      ( 비지니스관계자 )
   4. 구글폼생성하시고 
      수파베이스에 데이터삽입하는 폼 -> 폼아이디
   5. 구글앱스크립트 추가
      (수파베이스와 연동되는 구글폼설문조사.js)
      해서 실행한다. -> 사용자를 위한 폼( 시각적확인 )
   6.질문을 수파베이스에 넘기는 스크립트작성
      -> 수파베이스에서 확인하기
구글폼을 통해서 응답결과를 수파베이스 2개데이블로 저장로 보내고
수파베이스에서 구글시트응답결과/응시자로 보내야한다.

데이터무결성을 위한 이행적인 개발
